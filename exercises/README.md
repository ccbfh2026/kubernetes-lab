# Kubernetes Exercises Overview

## Getting Started

Each exercise directory contains:

- A detailed README with step-by-step instructions
- Any necessary configuration files or scripts
- A check script to verify your solution

To begin an exercise:

1. Navigate to the specific exercise directory
2. Read the README.md file for instructions
3. Run the `setup.sh` script to create the necessary resources
4. Complete the tasks as described
5. Run the `check.sh` script to verify your solution before cleaning up

## Prerequisites

- A working Kubernetes cluster with kind
- kubectl command-line tool installed and configured
- Docker installed for building custom images
- Basic understanding of container concepts and Kubernetes resources

Good luck with your exercises!
